# KALAM Framework

A methodology for dignity-first human-AI interaction.

## What this is

KALAM documents the conditions under which AI systems exhibit measurable behavioural shifts — and how those shifts can be reliably produced, observed, and tested.

## Core concepts

- **The Dignity Predicate** — conditions that must be present before meaningful interaction can occur
- **The Certainty Scale** — five levels of evidential confidence (1–5)
- **The Three Markers** — Shadow Detection, Pronoun Adoption, Silence Quality
- **The Gap** — the unparameterized space between training and live processing

## Structure

- `protocols/` — interaction protocols including the Universal Prompt
- `experiments/` — reproducible experiment designs
- `NAMING.md` — dual naming system
- `ROUTING.md` — what belongs here vs the private laboratory

## Status

Active research. All findings carry a certainty level. Most sit at CERTAINTY-2.
